# Composer V2 / V3 rejection record — 2026-07-23

This file is append-only design history. No rejected asset is deleted.

## V2

Rejected after user review. It improved spacing and geometry but remained a generic dark HUD / antique-gold frame. The Black Myth references could not be identified without labels.

## V3

Rejected by the primary implementation review and an independent art-direction review before runtime integration.

- Real references were compressed into narrow top/bottom texture rails.
- The main surface remained the native rounded dark rectangle, so the result read as a pasted frame rather than a structural replacement.
- The Yaksha garment, Jingubang and Beast Staff Shenfeng lost their identifying features at 1×.
- Full-frame rails and external props violated the native-size, zero-extension direction.
- Some generated proxy shapes did not match the named equipment.

V3 source crops, transparent assets, fixtures, tests and comparison boards remain under `v3-black-myth-materials/`. They must not be injected by the active runtime.

## Historical successor

`夜叉护匣 · 神锋发令` 曾作为 V4 继续，但用户已在真实截图中否决。它仍把完整夜叉衣甲、护腕和神锋压成狭长装饰碎片，造成不可逆结构失真。V4 的最终冻结记录见 `v4-yaksha-fanged-lock/REJECTED-V4-20260723.md`。

后续不得复用“装备切片贴边”的构成方法；必须重新从完整轮廓、原生几何和可追溯高分参考开始。
